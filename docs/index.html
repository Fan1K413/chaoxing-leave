GitHub Actions · 学习通 OA 自动化

自动提交请假与销假申请

一个公开仓库友好的自动化模板：脚本不保存 Cookie、定位、照片或个人预填信息，所有敏感参数都通过 Secrets / 环境变量注入。

模拟请假页面 查看 README GitHub 仓库

请假自动化

每周四北京时间 09:07 自动运行。

  • 请假时间：周五 12:00 至周日 18:00
  • 申请日期：运行当周周四
  • 面部拍照：通过 LEAVE_PHOTO_URL 注入
  • 脚本：leave.sh

模拟请假页面

用于在浏览器里预览请假表单、上传照片、手写签名并模拟提交；只做本地展示，不会调用真实提交接口，也不预置照片或签名图片。

  • 入口：docs/simulation
  • 自动缓存：姓名、学号、宿舍、辅导员、请假类型等非时间字段
  • 媒体缓存：上传的面部照片和手写签名会保存到当前浏览器
  • 时间字段：开始时间、结束时间、申请时间每次打开重新生成
  • 提交后：照片仍保持原网页的文件图标样式
  • 提交后的数据日志：可筛选上方详情表单字段；“审批操作”仅控制数据日志中的审批操作卡
  • 结构：页面已拆分为 HTML / CSS / JS，本地小图标资源内联在 CSS 中，便于维护

销假自动化

每周日北京时间 20:00 自动运行。

  • 销假时间：运行时北京时间
  • 定位：通过 CANCEL_LAT / CANCEL_LNG / CANCEL_ADDRESS 注入
  • 脚本：cancel.sh
  • 也支持在 Actions 页面手动触发

需要配置的 Secrets

进入 GitHub 仓库：Settings → Secrets and variables → Actions → New repository secret。

Secret 用途
CHAOXING_COOKIES 学习通 / 超星 OA 登录 Cookie。
LEAVE_PHOTO_URL 请假面部拍照图片链接。不要把照片文件提交到公开仓库。
CANCEL_LAT 销假定位纬度。
CANCEL_LNG 销假定位经度。
CANCEL_ADDRESS 销假定位地址。

本地运行：请假

CHAOXING_COOKIES='你的 Cookie' \
LEAVE_PHOTO_URL='https://example.com/photo.jpg' \
bash leave.sh

本地运行：销假

CHAOXING_COOKIES='你的 Cookie' \
CANCEL_LAT='纬度' \
CANCEL_LNG='经度' \
CANCEL_ADDRESS='地址' \
bash cancel.sh

执行流程

读取 Secrets
从 GitHub Actions Secrets 或本地环境变量读取 Cookie、照片链接和定位。
获取表单结构
实时请求 OA 表单结构、审批人和用户信息，避免维护固定模板。
构建并提交
按网页端格式构造表单数据后提交到 OA 保存接口。
公开模板仅保留自动化逻辑；敏感参数请始终放在 Secrets 中。
